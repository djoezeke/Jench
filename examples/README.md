# [JENCH](https://djoezeke.github.io/jench) [ EXAMPLES ]

This folder contains code examples for the C and C++ APIs.

> [!NOTE]
> Jench is still in development, **current version is incomplete and nonfunctional**.
> There is more room for improvements.

The purpose of Examples is to showcase integration, let you try Jench, and guide you toward
integrating Jench in your own application.

## Getting Started

Integration in a typical existing application, should take no time.

```cpp
#include <Jench/Jench.h>
```

Main resource:

- Read **[Documentation](https://djoezeke.github.io/jench/)** for detailed on how to integrate Jench.
